Characters have passive slots unlocking from 3 to 6 star. All passives are permanently locked once unlocked. The exact distribution of minor and major passives across star levels is not yet defined and depends on what the pools actually contain — that decision gets made once class and elemental passive content is designed.

What is fixed: class passives and elemental passives each contain both minor and major options. The 3 star unlock is always a class passive, coinciding with class lock. Beyond that the ordering is open.

Elemental awakening grants a small elemental passive at 2 star outside this system — see [[Elemental Awakening]]. Classes have a default identity passive that comes with the class lock at 3 star outside this system — see [[Class]].

See [[Class Passives]], [[Elemental Passives]].

## Pools

**Class pool** — drawn at 3 star and contributes to later draws. Contains both minor and major passives. Every class has its own curated list. See [[Class Passives]].

**Elemental pool** — available only to characters with an elemental awakening. Contains both minor and major passives. Characters without an awakening draw exclusively from the class pool for all passive slots. See [[Elemental Passives]].

**Broad pool** — draws from class and elemental pools combined. Used at later star levels. Curated per character combination rather than fully random, ensuring the result is coherent with what the character already is.
