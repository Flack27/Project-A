Class passives are drawn at 3 star and at later broad pool draws. Every class has its own curated pool containing both minor and major passives. The specific passive rolled adds individuality within the class while always being coherent with the role.

Non-elemental characters draw exclusively from this pool for all passive slots beyond 3 star. Elemental characters draw from both this pool and the elemental pool at later levels.

---

## Pool Structure

Each class has a curated list split into minor and major passives. Minor passives are smaller refinements — situational bonuses, stat edges, secondary effects. Major passives are the identity-defining ones that change how the character functions or scales.

_Full pools defined per class when the class list is finalised. Placeholder examples below._

**Warrior** — minor: bonus damage above HP threshold, increased crit rate on consecutive hits / major: stacking ATK on kill, execute damage below threshold

**Tank** — minor: reduced damage when at low HP, small shield on taking a critical hit / major: damage reduction while taunting, scaling DEF based on number of enemies targeting this character

**Support** — minor: extended buff duration, bonus effect on cleanse / major: heal scaling with SPD, group shield on using a heal skill

**Debuffer** — minor: bonus damage against debuffed targets, reduced debuff cooldowns / major: increased effect accuracy, extended debuff duration on all applied effects
