All-rounder damage dealer built around a stack and payoff rhythm. Basic attacks and single target skills build stacks, an AoE burst finisher expends them for scaling damage output. ATK and crit investment maximises burst size on the payoff, HP and DEF investment sustains through longer fights to hit the payoff repeatedly — both directions use the same mechanic. Chain on kill extends the action window after a successful burst, feeding immediately back into stack generation. Execute scaling on missing HP handles priority single target kills between burst windows. No dramatic stat trade-offs, no hard counters — the broadest weapon access of any physical class reflects the all-rounder identity.
